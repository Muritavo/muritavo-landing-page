# Pré requisitos
1. Instalar o nodejs mais recente
> Aqui eu sugiro utilizar NVM, que nos auxilia no gerenciamento de diversas verões do NodeJS
2. Instalar o yarn
```bash
npm i -g yarn
```
3. Instalar as dependências do projeto
```bash
yarn
```

# Como testar

Rode o comando:
```bash
yarn test:watch
```
